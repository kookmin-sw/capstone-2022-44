## 1. 프로잭트 소개

### 프로젝트 목적

* 사용자의 발표 자료가 청취자의 예상되는 시각 흐름과 청취자의 실제 시각 흐름을 시각화하여 사용자에게 보여주고 시각화한 자료의 분석을 통해 
  발표 자료 배치를 추천해준다.

            
### 주요 기능

* webgazer.js를 통해 웹페이지에서 eye tracking을 통해 사용자가 올린 pdf파일의 시선을 분석
* PPT 슬라이드 별 사용자와 평가자의 eye tracking 데이터 비교 분석
